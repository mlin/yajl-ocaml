Module JSON.Ops


module Ops: sig .. end
Convenient operators and functions for working with the JSON representation. The author tends open JSON.Ops in programs.
exception JSON_type_mismatch of string * JSON.t
Raised when one of the operators/functions below expects a certain JSON type but is given another. Carries the expected type (e.g. "Object", "Float") and the actual value given.
val ($) : JSON.t -> string -> JSON.t
obj $ "foo" retrieves the value of key "foo" in the object obj.
exception JSON_not_found of string * JSON.t
Raised by the operator ($) when the given JSON is an object but does not contain the expected key. Carries the key and the object.
val ($?) : JSON.t -> string -> bool
obj $? "foo" tests whether object obj has key "foo".
val ($!) : JSON.t -> string * JSON.t -> JSON.t
obj $! ("foo",json) returns an object in which the key "foo" has been set to the value json, replacing a previous value at that key if one was present. The original object is not modified.
val ($@) : JSON.t -> int -> JSON.t
arr $@ k retrieves the value at zero-based position k in the array arr.
val ($@!) : JSON.t -> int * JSON.t -> JSON.t
arr $@! (k,json) returns an array in which zero-based position k has been replaced with json. The original array is not modified.
val json_bool : JSON.t -> bool
val json_int : JSON.t -> int
val json_float : JSON.t -> float
val json_float' : JSON.t -> float
val json_number : JSON.t -> [ `Float of float | `Int of int ]
val json_string : JSON.t -> string
val json_object : JSON.t -> (string, JSON.t) Batteries_uni.Map.t
val json_object_keys : JSON.t -> string list
val json_array : JSON.t -> JSON.t Batteries_uni.Vect.t
val json_array_length : JSON.t -> int
Accessors, any of which may raise JSON_type_mismatch. json_float' also accepts `Int values and coerces them to float.
