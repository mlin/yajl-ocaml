Index of values


($) [JSON.Operators]
obj $ "foo" retrieves the value of key "foo" in the object obj.
($+) [JSON.Operators]
obj $+ ("foo",json) returns an object in which the key "foo" has been set to the value json, replacing a previous value at that key if one was present.
($-) [JSON.Operators]
obj $- "foo" returns an object in which the key "foo" has been removed if it existed.
($?) [JSON.Operators]
obj $? "foo" tests whether object obj has key "foo".
($@!) [JSON.Operators]
arr $@! (k,json) returns an array in which zero-based position k has been replaced with json.
($@) [JSON.Operators]
arr $@ k retrieves the value at zero-based position k in the array arr.
